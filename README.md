# 📝 Microsoft Registration Portal

A simple and responsive **registration form website** built using **HTML and CSS**. This project demonstrates a clean UI layout with a header, navigation bar, hero section, form, and footer.

---

## 🚀 Features

* ✅ Clean and modern UI design
* ✅ Fully responsive layout
* ✅ Navigation bar with hover effects
* ✅ Hero image section
* ✅ User registration form
* ✅ Styled input fields with focus effects
* ✅ Call-to-action submit button
* ✅ Footer section

---

## 🛠️ Technologies Used

* HTML5
* CSS3 (Custom Styling)
